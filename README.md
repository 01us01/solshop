# 이우솔 쇼핑몰 개인 프로젝트

개요
 - 쇼핑몰의 주요기능인 상품목록, 장바구니, 게시판, 로그인, 관리자 로그인 등을 구현한 사이트 입니다.

주요기술
 - 웹 화면 구성 : HTML, CSS, Jquery, JavaScript
 - 프로그램 언어 : JAVA, Servlet, JSP
 - DB : JDBC
 - 개발 툴 : Eclipse, Apache Tomcat v8.5, Spring
